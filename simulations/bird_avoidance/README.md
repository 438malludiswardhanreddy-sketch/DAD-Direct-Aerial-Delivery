# Bird Avoidance Scenario

## 1. Scenario Objective
Demonstrate dynamic path corrections and sharp yaw angle overrides when a fast-moving object (bird) enters the flight vector range.

## 2. Input Conditions
*   **Altitude**: 45.0m AGL
*   **Cruise Speed**: 12.5 m/s
*   **Bird Proximity Event**: Detected at 6.5 metres distance, 12.0° azimuth angle.
*   **Failsafe Action**: Steer opposite to threat azimuth.
