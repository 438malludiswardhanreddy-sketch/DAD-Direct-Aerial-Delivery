# Wire Avoidance Scenario

## 1. Scenario Objective
Demonstrate vertical flight path corrections when thin horizontal power lines or telephone wires are detected directly ahead.

## 2. Input Conditions
*   **Altitude**: 45.0m AGL
*   **Cruise Speed**: 12.5 m/s
*   **Wire Proximity Event**: Detected at 9.0 metres range.
*   **Failsafe Action**: Climb +3.5m above current altitude.
