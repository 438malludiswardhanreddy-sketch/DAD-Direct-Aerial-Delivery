# Urban Delivery Scenario

## 1. Scenario Objective
Demonstrate a normal, autonomous flight from Launch Station A to Landing Hub B in an urban canyon environment.

## 2. Input Conditions
*   **Start Coordinates**: 17.6590, 75.9059
*   **Target Coordinates**: 17.6721, 75.9125
*   **Altitude**: 45.0m AGL (Above Ground Level)
*   **Cruise Speed**: 12.5 m/s
*   **Battery Starting State**: 100%
*   **Weather**: Clear sky, 4.2 m/s NW wind.
