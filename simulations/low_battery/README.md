# Low Battery Scenario

## 1. Scenario Objective
Demonstrate the Return-to-Home (RTH) failsafe mechanism when telemetry detects battery levels dropping to 20%.

## 2. Input Conditions
*   **Cruise Altitude**: 45.0m AGL
*   **Speed**: 12.5 m/s
*   **Battery Status**: Drops from 21.0% to 20.0% during flight.
*   **Failsafe Action**: Abort current mission and fly back to the launching station immediately.
