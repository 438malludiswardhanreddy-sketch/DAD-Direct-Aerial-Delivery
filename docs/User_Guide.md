# DAD Operations User Guide

## 1. Booking a Delivery
1.  Open the customer mobile app screen.
2.  Input destination coordinates (Latitude/Longitude).
3.  Select payload type and submit.
4.  Track the drone position on the live map widget.

## 2. Telemetry Supervision
1.  Launch the FastAPI server and open `dashboard/index.html` in a browser.
2.  Review fleet cards. Hover states show real-time battery percentages.
3.  Verify weather warnings.
4.  In case of emergency, click the **Abort Mission** button to send immediate RTL commands.
